# Changelog

## [1.0.0] - 2025-11-10

### Added
- Initial release of VSCode Theme Map Plugin
- Core functionality for extracting and mapping VS Code theme variables
- Tailwind CSS plugin for dynamic theme integration
- Shadcn UI theme variant support
- Comprehensive unit test suite
- Continuous Integration (CI) workflow
- Detailed documentation and contribution guidelines

### Features
- Dynamic theme variable extraction
- Intelligent theme mapping for Tailwind CSS
- Flexible configuration options
- Minimal runtime overhead
- Framework-agnostic design

### Core Components
- `VSCodeThemeVariableExtractor`: Theme variable detection
- `ThemeMapper`: Theme variable mapping logic
- `vsCodeTailwindPlugin`: Tailwind CSS plugin for theme integration
- `createShadcnThemeVariants`: Shadcn UI theme variant generator

### Development
- TypeScript implementation
- Jest for unit testing
- ESLint for code quality
- GitHub Actions for CI/CD
